# Portafolio CV - Estructura base

Estructura inicial para una landing page de CV inspirada en el estilo minimalista de referencia.

## Estructura

- `index.html`: layout principal de la landing.
- `css/styles.css`: estilos visuales y responsive.
- `js/main.js`: renderizado dinamico del contenido.
- `data/profile.js`: datos editables de perfil, skills, experiencia y contacto.
- `assets/images/`: carpeta para foto de perfil, iconos o capturas.

## Como editar contenido

1. Abre `data/profile.js`.
2. Reemplaza los textos placeholder con tu informacion real.
3. Si quieres mas tarjetas de skills o experiencia, agrega objetos en los arrays correspondientes.

## Como visualizar

Abre `index.html` en el navegador.
